# PRODIGY_EMS_02
<h2><b>Overview</b></h2>
This project focuses on developing a web-based Employee Management System to efficiently manage and maintain employee records. The system allows users to perform essential operations such as adding, viewing, updating, and deleting employee details through a user-friendly interface.

The <b>backend</b> is implemented using JSP and JDBC to handle business logic and ensure secure database communication with PostgreSQL.<br>
The <b>frontend</b> is developed using HTML, CSS, and Bootstrap, providing a responsive and intuitive UI for smooth interaction. Dynamic data rendering ensures real-time updates of employee information.

This individual project enhanced practical understanding of CRUD operations, database connectivity, and full-stack web application development, making it a valuable hands-on learning experience.

<h2><b>Technologies</b></h2>
Technologies Used for Employee Management System : JSP, JDBC, PostgreSQL, HTML, CSS, Bootstrap
<b><h2>Team Size</b></h2>1 (Individual Project)
<h2><b>Conclusion</b></h2>
The Employee Management System successfully provides a web-based solution for managing employee records with full CRUD functionality. This project strengthened skills in JSP, JDBC, PostgreSQL, and responsive UI development, offering valuable full-stack development experience.
